# CLAUDE.md

This repository contains a fork of the QMK repository.

## What is QMK Firmware?
QMK (Quantum Mechanical Keyboard) is an open source community centered around developing computer input devices. The community encompasses all sorts of input devices, such as keyboards, mice, and MIDI devices.

Documentation for this project can be found at https://docs.qmk.fm/

## Limitations of Scope
This fork is only concerned with building firmware for Aurora keyboards. These keyboards firmwares are located in `keyboards/splitkb/aurora`.

## Building
Build firmware with command:
`./build.sh splitkb/aurora/sweep qwerty_hrm`
